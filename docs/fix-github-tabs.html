Correção de Abas para GitHub Pages

Este guia explica como corrigir o problema de sobreposição das abas na versão GitHub Pages.

O problema ocorre quando o conteúdo de uma aba (como "Restrições de Tráfego") aparece indevidamente em outra aba (como "Relatório" ou "Eventos").

Passo 1: Substituir o container de abas

Localize o código do container de abas no arquivo HTML e substitua pelo seguinte:

<!-- NOVO: Tabs fixas na parte inferior do layout (inicialmente minimizadas) --> <div class="bottom-tabs-container minimized"> <!-- Navegação das abas inferiores --> <div class="bottom-tabs-nav"> <button class="bottom-tab-btn active" data-tab="bottom-events" onclick="showTab('bottom-events-content')">Eventos na Rota</button> <button class="bottom-tab-btn" data-tab="bottom-restrictions" onclick="showTab('bottom-restrictions-content')">Restrições de Tráfego</button> <button class="bottom-tab-btn" data-tab="bottom-report" onclick="showTab('bottom-report-content')">Relatório da Rota</button> </div> <!-- Container das abas com rolagem independente para cada aba --> <div class="bottom-tabs-content-container"> <!-- Conteúdo da aba Eventos --> <div class="bottom-tab-content active" id="bottom-events-content" style="display: block;"> <h5>Eventos nas cidades da rota:</h5> <div class="events-list" id="events-list"> <p class="text-muted">Nenhum evento encontrado. Adicione locais e otimize a rota para ver eventos nas cidades do percurso.</p> </div> </div> <!-- Conteúdo da aba Restrições --> <div class="bottom-tab-content" id="bottom-restrictions-content" style="display: none;"> <h5>Restrições de tráfego para caminhões:</h5> <div class="restrictions-list" id="restrictions-list"> <p class="text-muted">Nenhuma restrição encontrada. Adicione locais e otimize a rota para ver as restrições de tráfego.</p> </div> </div> <!-- Conteúdo da aba Relatório --> <div class="bottom-tab-content" id="bottom-report-content" style="display: none;"> <h5>Resumo da rota otimizada:</h5> <div class="route-info p-3 border rounded mb-3 bg-light" id="route-info"> <p class="text-muted">Otimize a rota para ver o relatório detalhado.</p> </div> <h5>Sequência de entrega:</h5> <div class="route-sequence border rounded p-3" id="route-sequence"> <p class="text-muted">A sequência de rota será exibida aqui após a otimização.</p> </div> </div> </div> </div>

Passo 2: Adicionar função de controle de abas

Adicione esta função JavaScript antes do fechamento da tag </body>:

<script> // Função simplificada para controlar as abas function showTab(tabContentId) { // Esconder todos os conteúdos de abas var allTabContents = document.querySelectorAll('.bottom-tab-content'); allTabContents.forEach(function(content) { content.style.display = 'none'; content.classList.remove('active'); }); // Mostrar o conteúdo da aba selecionada var selectedContent = document.getElementById(tabContentId); if (selectedContent) { selectedContent.style.display = 'block'; selectedContent.classList.add('active'); } // Atualizar classes ativas nos botões var allTabButtons = document.querySelectorAll('.bottom-tab-btn'); allTabButtons.forEach(function(button) { button.classList.remove('active'); // Adicionar classe active ao botão correspondente if (button.getAttribute('data-tab') === tabContentId.replace('-content', '')) { button.classList.add('active'); } }); // Expandir o container se estiver minimizado var tabsContainer = document.querySelector('.bottom-tabs-container'); if (tabsContainer && tabsContainer.classList.contains('minimized')) { tabsContainer.classList.remove('minimized'); } } // Verificar e corrigir após otimização de rota document.addEventListener('DOMContentLoaded', function() { var optimizeButton = document.getElementById('optimize-route'); if (optimizeButton) { optimizeButton.addEventListener('click', function() { // Aguardar a conclusão da otimização setTimeout(function() { // Corrigir a exibição das abas após otimização var reportTabButton = document.querySelector('.bottom-tab-btn[data-tab="bottom-report"]'); if (reportTabButton) { reportTabButton.click(); } }, 2000); }); } }); </script>

Passo 3: Modificar o evento de clique em "Otimizar Rota"

Localize a função que trata o clique no botão "Otimizar Rota" e adicione o seguinte código ao final:

// Navegar para a aba de relatório const reportTabButton = document.querySelector('.bottom-tab-btn[data-tab="bottom-report"]'); if (reportTabButton) { reportTabButton.click(); // Garantir exibição adequada do conteúdo showTab('bottom-report-content'); }
