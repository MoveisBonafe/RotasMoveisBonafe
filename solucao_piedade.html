Solução para o problema da cidade de Piedade

Problema identificado: A cidade de Piedade está aparecendo com informações duplicadas e datas incorretas:

  • Data incorreta: Piedade | 18/03/2025
  • Descrição incorreta: Aniversário de fundação de Piedade em 19/03/1842
  • Dados corretos: data 20/05/2025 e fundação em 20/05/1842

Solução Implementada

Criamos um script específico que corrige as datas incorretas e remove as entradas duplicadas. Este script:

  1. Corrige a data de aniversário de Piedade para 20/05/2025
  2. Corrige a descrição para "Aniversário de fundação de Piedade em 20/05/1842"
  3. Remove entradas duplicadas da cidade
  4. Corrige os dados no modelo global (JavaScript) e nos elementos do DOM

Como usar a solução

Para implementar esta solução, você precisa:

  1. Copiar o arquivo city-events-fix.js para seu repositório
  2. Adicionar a seguinte linha ao final de seu arquivo index.html, logo antes da tag </body>: 
    <script src="city-events-fix.js"></script>

Solução Permanente

Para uma solução mais permanente, também corrigimos os dados na fonte, removendo entradas duplicadas de Piedade no arquivo docs/city-events-data.js e garantindo que todas as instâncias usem a data correta 20/05/2025.

Isso deve resolver tanto o problema de duplicação quanto o problema da data incorreta.
