PÁGINA SELETORA DE VERSÕES
=========================

Para resolver o problema de forma definitiva, criamos uma página seletora de versões:

1. index-version-selector.html

Esta página:
- Exibe todas as versões disponíveis
- Destaca a versão recomendada (standalone-auto.html)
- Permite que o usuário selecione a versão mais adequada

COMO USAR:
1. Faça upload de todos os arquivos para seu repositório
2. Renomeie index-version-selector.html para index.html
3. Visite a página inicial do seu GitHub Pages

Dessa forma, os usuários poderão escolher a versão que funciona melhor, e 
caso ainda haja problemas com uma versão específica, eles podem experimentar as outras.

Esta abordagem é mais amigável e robusta para lidar com diferentes configurações do GitHub Pages.
